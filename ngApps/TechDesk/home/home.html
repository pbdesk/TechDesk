Getting Started

New to Angular.js? Before getting into Angular Material, it might be helpful to read about the framework.

How do I start?

  • Fork a Codepen
  • Clone a Github Starter Project

Including Angular Material and its dependencies

Contributing to Angular Material

  • To contribute, fork our GitHub repository.
  • Please read our contributor guidelines.
  • For problems, search the issues and/or create a new issue .
  • For questions, search the forum and/or post a new message.
